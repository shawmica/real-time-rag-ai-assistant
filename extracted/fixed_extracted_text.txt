       What is Boosting? 
Before XGBoost, understand Boosting concept. 
 
  Basic Idea of Boosting 
Instead of training one big model: 
     Train many weak models (small trees) 
     Each new model learns from the previous 
model’s errors 
     Combine all models → strong final model 
 
Step-by-step intuition: 
1. First tree predicts. 
2. Calculate errors (residuals). 
3. Second tree tries to correct those errors. 
4. Repeat many times. 
5. Add all predictions together. 
Final Prediction = Sum of all small trees. 
 
   Why Boosting is Powerful? 
• Focuses on difficult samples. 
• Reduces bias. 
• Very strong predictive power.

   What it is

   Common ways leakage happens

o scaler.fit(X) before splitting → test

• Feature engineering using future info

• Target leakage 
o Feature is directly/indirectly created

o Trying multiple models/hyperparams
