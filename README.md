# Real-Time RAG AI Assistant

A production-style Retrieval-Augmented Generation (RAG) AI assistant built using:

- LangChain
- ChromaDB
- FastAPI
- Streamlit
- HuggingFace Embeddings
- Groq LLM

## Features

- PDF Upload & Ingestion
- Semantic Search
- Context-Aware Responses
- Vector Database Retrieval
- Real-Time AI Chat Interface

## Tech Stack

Python • LangChain • ChromaDB • FastAPI • Streamlit • Groq • HuggingFace

## Architecture

User → Streamlit → FastAPI → Retriever → ChromaDB → Groq LLM → Answer
